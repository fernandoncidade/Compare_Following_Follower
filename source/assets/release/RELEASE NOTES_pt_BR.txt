NOTAS DE LANÇAMENTO — Compare - Following and Follower
Versão: 2026.2.29.0
Data: 29 de fevereiro de 2026

Resumo da versão
Esta versão melhora a estabilidade do uso diário, deixa o fluxo de token mais discreto e torna o preenchimento de usuário mais inteligente.

Melhorias principais
- O reinício automático após o reset foi reforçado para funcionar com mais confiabilidade.
- A definição e o reset do token agora acontecem de forma silenciosa, sem abrir janelas de comando.
- O campo de usuário passa a iniciar vazio quando não existe usuário salvo.
- O aplicativo agora lembra o último usuário informado e carrega esse valor nas próximas aberturas.
- Quando um novo usuário é informado, ele substitui automaticamente o usuário salvo anteriormente.
- Ao executar "Resetar Token/Variáveis de Ambiente", o usuário salvo também é apagado e o campo de usuário é limpo.

Melhorias de experiência
- O reset ficou mais completo, removendo também informações usadas no preenchimento automático.
- As mensagens de resultado ficaram mais claras sobre o que foi limpo.

Compatibilidade e dados
- Não é necessária migração manual.
- As melhorias são aplicadas automaticamente após a atualização.

NOTAS DE LANÇAMENTO — Compare - Following and Follower
Versão: 2026.2.28.0
Data: 28 de fevereiro de 2026

Resumo da versão
Esta versão foca em facilitar a configuração de acesso ao GitHub e em dar mais controle sobre os dados locais do aplicativo, com fluxos guiados por confirmação.

Novidades principais
- Novo fluxo para definir o token do GitHub diretamente pela interface.
- Salvamento do token em variáveis de usuário e de sistema no Windows (quando permitido).
- Novo recurso para resetar token e variáveis de ambiente pelo menu Configurações.
- No reset, o aplicativo também remove os dados locais relacionados e reinicia automaticamente.
- Nova ação "Excluir" no menu Arquivo para apagar os arquivos do banco local com confirmação.
- A janela de exclusão agora mostra somente os arquivos que realmente existem.
- Quando não houver arquivos locais, o aplicativo informa isso claramente antes de qualquer ação.

Melhorias de usabilidade
- Novos atalhos para ações frequentes:
  - Ctrl+Shift+D (Excluir banco local)
  - Ctrl+Shift+T (Definir token GitHub)
  - Ctrl+Shift+R (Resetar token/variáveis)
- Mensagens de sucesso, aviso e erro mais diretas durante configuração, reset e exclusão.
- Fluxo mais assistido quando o token estiver ausente: o app pode solicitar o token e continuar o processo atual.

Melhorias de confiabilidade
- O aplicativo aproveita melhor tokens já salvos no sistema, mesmo quando a sessão atual não tinha carregado a variável.
- Em cenários com permissão parcial, o app informa detalhes do que foi concluído para evitar dúvidas.

Compatibilidade e dados
- O formato dos dados locais foi preservado.
- Não há necessidade de migração manual para continuar usando a aplicação.

NOTAS DE LANÇAMENTO — Compare - Following and Follower
Versão: 2026.2.26.0
Data: 26 de fevereiro de 2026

Resumo da versão
Esta versão marca a consolidação do projeto Compare - Following and Follower como um app focado em comparação de relacionamentos no GitHub, com fluxo de uso mais completo, atalhos úteis e experiência de interface mais organizada.

Novidades principais
- Novo menu Arquivo com ações:
  - Novo: limpa os dados exibidos e reseta os arquivos locais de cache/estado.
  - Abrir: importa arquivos .json compatíveis com o formato do aplicativo.
  - Salvar: exporta dados consolidados em .json.
  - Ajuda: exibe a lista de atalhos disponíveis.
  - Fechar: encerra o aplicativo.

- Estrutura de menus refinada:
  - Configurações agora concentra apenas Idioma.
  - Opções mantém Manual e Sobre.
  - Organização mais intuitiva para navegação diária.

- Atalhos de teclado implementados/ajustados:
  - Ctrl+N (Novo)
  - Ctrl+O (Abrir)
  - Ctrl+S (Salvar)
  - F1 (Ajuda de atalhos)
  - Ctrl+Q (Fechar)
  - Alt+I (Idioma)
  - Ctrl+Shift+M (Manual)
  - Ctrl+Shift+A (Sobre)

- Importação e exportação de dados:
  - Validação de estrutura JSON para evitar carregamentos inválidos.
  - Suporte a pacote com cache + state em um único arquivo.
  - Mensagens claras de sucesso e falha durante persistência.

- Traduções e internacionalização:
  - Inclusão das chaves de tradução faltantes em pt-BR e en-US.
  - Cobertura de textos de menu, mensagens operacionais e chaves auxiliares do Sobre.

Melhorias de usabilidade
- Fluxo de "Novo/Abrir/Salvar" mais previsível e orientado por confirmação.
- Diálogo de ajuda de atalhos ampliado com ações realmente disponíveis no app.
- Mensagens de erro e status mais consistentes para ações de arquivo.

Ajustes de comportamento de janelas
- Manual e Sobre continuam independentes para permitir foco da janela principal.
- Encerramento da GUI principal fecha automaticamente as janelas auxiliares.

Compatibilidade e dados
- Arquivos locais continuam compatíveis com o formato do projeto.
- Estrutura persistente de cache/estado preservada.
- Sem migração manual necessária.

Próximos passos sugeridos
- Evoluir exportação para incluir snapshots versionados por data/hora.
- Adicionar comparação visual entre dois arquivos exportados.
- Expandir indicadores para métricas históricas de crescimento de rede.

Nota final
Se esta versão fosse uma história, seria o capítulo em que o app deixa de ser "ferramenta útil" e vira "companheiro de auditoria social no GitHub".
Menos caos, mais contexto.
